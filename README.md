# GLiM 
**Generative Lightweight Interaction Manifest**

GLiM is a state-driven development framework for Claude 4.6. It transforms the often-unpredictable nature of AI coding into a disciplined, RPG-inspired system where every architectural decision is grounded in "Stats" and "Logic."

## Installation

To add GLiM to your project, run the following command in your terminal using Claude Code:

```bash
npx skills add [https://github.com/Herorif/GLiM](https://github.com/Herorif/GLiM)
